# Colin Builds AI

Practical AI projects, honest testing, and repeatable build systems.

This repository is the public source library for the **Colin Builds AI** channel. Each project is designed to be understandable, reproducible, and useful—not just another demo held together by thumbnail glue.

## Projects

### AI Trend Radar

A local Python tool that finds rising AI repositories, Hugging Face projects, and Hacker News discussions, then ranks them using transparent scoring and historical momentum.

- Source: [`projects/ai-trend-radar`](projects/ai-trend-radar)
- Demo dashboard: [`projects/ai-trend-radar/examples/demo-report.html`](projects/ai-trend-radar/examples/demo-report.html)

## Principles

- Build before making claims.
- Show what works and what breaks.
- Keep scoring and decisions explainable.
- Use public or synthetic data in public examples.
- Never commit credentials or private information.
